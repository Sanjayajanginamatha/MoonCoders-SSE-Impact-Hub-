# SSE Impact Hub
SSE Impact Hub is a Social Stock Exchange-inspired platform that ensures transparent and accountable NGO funding.

## Features
* NGO verification system
* Project-based funding
* Donation tracking
* Fund transparency
* 80G certificate generation
* Blockchain-based audit trail

## Tech Stack
* Frontend: React
* Backend: Spring Boot
* Database: PostgreSQL

## Goal
To ensure every rupee donated is traceable, transparent, and impactful.
